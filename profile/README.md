## gRPC ecosystem

[thesis/grpc](https://github.com/thesis-php/grpc) — non-blocking full-featured gRPC implementation for PHP.

[thesis/protobuf](https://github.com/thesis-php/protobuf) — a modern strictly typed full-featured serializer for Google's protocol buffers.

[thesis/protoc-plugin](https://github.com/thesis-php/protoc-plugin) — a php plugin for protobuf compiler (protoc).

## Queue drivers

[thesis/amqp](https://github.com/thesis-php/amqp) — pure non-blocking strictly typed full-featured PHP driver for AMQP 0.9.1 protocol.

[thesis/nats](https://github.com/thesis-php/nats) — pure non-blocking strictly typed full-featured PHP driver for NATS.

[thesis/pgmq](https://github.com/thesis-php/pgmq) — a non-blocking php client for Postgres Message Queue (PGMQ) extension.

## Time

[thesis/time-span](https://github.com/thesis-php/time-span) — time capabilities library.
